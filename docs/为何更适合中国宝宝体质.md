# 为何更适合中国宝宝体质

lingti-bot 更适合中国用户的几个原因：

---

## 隐私声明

> **我们承诺：一键集成平台（微信公众号「灵缇小秘」等）严格保护用户隐私，不保存任何用户数据。**

### 数据完全本地化

| 数据类型 | 存储位置 | 云端是否保存 |
|---------|---------|:-----------:|
| 对话历史 | 用户本地 | **否** |
| 上下文记忆 | 用户本地 | **否** |
| 配置文件 | 用户本地 | **否** |
| API 密钥 | 用户本地 | **否** |

### 云中继的工作原理

```
┌─────────────┐      ┌─────────────────┐      ┌─────────────┐
│  微信/飞书   │ ───▶ │  云中继网关      │ ───▶ │  本地 Bot   │
│  用户消息    │      │  (仅转发，不存储) │      │  (处理+存储) │
└─────────────┘      └─────────────────┘      └─────────────┘
                            │
                            ▼
                     纯路由转发
                     不记录内容
                     不保存历史
                     无状态运行
```

**云中继平台仅作为 AI 路由网关使用**：
- 消息实时转发，转发后立即丢弃
- 不记录、不分析、不存储任何对话内容
- 用户的所有数据（对话历史、记忆、配置）均保存在用户自己的电脑上
- 即使云中继服务停止，用户数据也不会丢失

---

## 1. 国内平台全面支持

| 平台 | 状态 |
|------|:----:|
| 飞书/Lark | ✅ 已支持 |
| 钉钉 | ✅ 已支持 |
| 企业微信 | ✅ 已支持 |
| 微信公众号 | ✅ 已支持（云中继）|

而非只支持 Slack/Discord 等海外平台。

## 2. 中文语音输入默认

- whisper 默认 `-l zh` 中文识别
- 不需要额外配置语言
- 开箱即用的中文语音交互

## 3. 云中继模式

- 解决国内网络环境问题
- WebSocket 穿透，无需公网 IP
- 微信等平台无需自建服务器
- **隐私优先**：仅路由转发，不存储任何数据

## 4. 本地化日历集成

- 修复了 macOS 日历的中文日期格式问题
- 使用日期构造器而非依赖系统 locale
- 完美支持中文系统环境

## 5. 中文文档

- README 全中文
- 命令示例贴合国内使用场景
- 无需翻译即可上手

## 6. 一键安装

```bash
curl -fsSL https://cli.lingti.com/install.sh | bash -s -- --bot
```
